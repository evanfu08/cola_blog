# COLA 博客正式上线！🎉

欢迎来到我的个人博客！经过精心定制和优化，这个基于 Next.js 的博客终于与大家见面了。

## 博客特色

### 🎨 浅海配色主题
采用清新的浅海风格配色方案：
- 主色调：海洋蓝 (#4A90E2)
- 辅助色：浅青色 (#7EC8E3)
- 背景：浅蓝渐变系列

整体视觉效果清爽舒适，给人以海洋般的宁静感。

### 🎵 音乐播放器
首页集成了功能完整的音乐播放器：
- **当前曲目**：英雄 - 艾志恒Asen
- **功能特性**：
  - 实时进度条（支持点击/拖动跳转）
  - 播放/暂停控制
  - LRC 歌词同步显示（右下角 Toast 弹窗）
  - 封面展示

### 📹 视频展示
首页视频卡片自动播放，展示个性化内容。

### ⚙️ 强大的配置系统
通过右上角齿轮图标可以访问配置面板，包含：
- **网站设置**：站点信息、社交链接、图片管理
- **色彩配置**：7 种预设配色方案（浅海、春暖、秋实、深夜、极光、森林、日落）
- **首页布局**：拖拽调整卡片位置和大小
- **媒体配置**：上传音乐和视频文件

### 🔐 安全的内容管理
使用 GitHub App 进行权限控制：
- 只有拥有私钥的用户才能修改内容
- 所有修改通过 GitHub API 提交
- Vercel 自动部署更新

## 技术栈

- **框架**: Next.js 16 + React 19
- **样式**: Tailwind CSS
- **动画**: Motion (Framer Motion)
- **部署**: Vercel
- **内容管理**: GitHub API + GitHub App
